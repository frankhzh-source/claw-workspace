### SVG 多页打印修复（方案 B：HTML 包裹）
- 用户反馈 Chrome 打印 SVG 突然变成缩到一页（昨天之前正常），怀疑是 Chrome 更新导致
- 诊断：Chrome 把独立 .svg 文件当图片处理，图片天生不分页；旧版本可能默认当文档处理
- 方案 B：将 SVG 嵌入 HTML，用 `viewBox` 切片 + `page-break-after: always` + `overflow:hidden`
- 结果：3 个 div，每个展示原 SVG 的 1130px 高度截面，打印时各占 1 页 A4
- 对原 SVG 无影响，不影响编辑，打印时改用 HTML 文件
