①Prompt 转化
真实的 Prompt 模板不直接暴露。Agent 层从多维度属性动态组装 flat text，无法逆向还原结构。
②结果缓存
按内容 hash + 参数 hash 缓存。命中直接返回不调 API。三重收益：减少暴露 + 降成本 + 降延迟。
③多模型路由
抽象接口同时跑 3+ 家模型。最大谈判筹码："你的表现在我这边排名第 X" + 防止锁定。
④请求脱敏
Logo/品牌名/条形码 → 自动模糊。客户 ID → hash。分类 → 通用类目名。不发原始数据。
4可共享 vs 不可共享
✓放心共享（建立信任）
可行业使用场景描述"某品牌需要 500 个 SKU 视频"
可视频质量反馈"背景太复杂，主体不突出"
可行业普遍痛点"电商客户普遍反映背景一致性差"
可竞品差距对比"X 模型在某场景下比 VIDU 好 30%"
可脱敏测试截图去除品牌信息和客户标识
✕红线：绝不共享
禁客户名单和合同⭐ 被绕过就归零
禁完整 Prompt 模板⭐ 拿去做预设模板
禁工作流编排源码⭐ 节点逻辑是决策方式
禁质量评估量化标准⭐ 判断力比生成力更稀缺
禁客户付费意愿和预算⭐ 了解后可绕过你定价
