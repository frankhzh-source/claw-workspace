## 三、分维度解读

### 1. 角色一致性（人物/服装不变形）

| 评级 | 模型 |
|------|------|
| ★★★★★ | Seedream 4.5（LibTV 官方标注多角色一致性） |
| ★★★★☆ | Seedream 4.6、Lib Navo Pro |
| ★★★☆☆ | Seedream 5.0 Lite、Seedream 4.0、Lib Navo 2、Qwen Image、悠船 V7/V8.1 |
| ★★☆☆☆ | Lib Image、Lib Navo、Z-image Turbo、悠船 Niji 7 |

**关键结论**：
- 电商人物一致性首选 **Seedream 4.5**（官方标注就是多角色一致性）。
- 如果追求更高画质且能接受一致性略降，选 **Seedream 4.6**。
- 悠船系列出图质感高，但人物一致性更依赖 Reference 技巧，不如 Seedream 原生稳定。

### 2. 服装/材质表现（家居服面料、印花、褶皱）

| 评级 | 模型 |
|------|------|
| ★★★★★ | Seedream 4.6、悠船 V7/V8.1 |
| ★★★★☆ | Seedream 4.5、Seedream 4.0、Lib Navo Pro、Lib Navo 2、Qwen Image |
| ★★★☆☆ | Seedream 5.0 Lite、Lib Image、Lib Navo |
| ★★☆☆☆ | Z-image Turbo、悠船 Niji 7 |

**关键结论**：
- 缎面、丝绸、棉质家居服的质感，**Seedream 4.6** 和 **悠船 V8.1** 最强。
- Seedream 4.5 在中式印花、传统纹样上更有优势。

### 3. 中文语义理解（提示词准确还原）

| 评级 | 模型 |
|------|------|
| ★★★★★ | Seedream 4.5、Seedream 4.6 |
| ★★★★☆ | Seedream 5.0 Lite、Seedream 4.0、Qwen Image、Qwen Edit |
| ★★★☆☆ | Lib Image、Lib Navo 系列、悠船 V7/V8.1/Niji 7、Z-image Turbo |

**关键结论**：
- 中文提示词理解 **Seedream 系列** 最好，其次是 **Qwen Image**（阿里系本土中文强）。
- 悠船/Midjourney 用中文提示词容易丢细节，建议用英文或中英双语。

## 五、使用策略建议

1. **电商人物一致性优先：Seedream 4.5 打底 + 参考图锁脸**  
   用维度1主视觉肖像作为参考图，4.5 在多角色一致性上原生最强。

2. **高质感服装材质：切换到 Seedream 4.6 或悠船 V8.1**  
   同一个 Prompt 在不同模型下表现不同，材质特写可以单独切模型。

3. **不要全链路用同一个模型**  
   主视觉用 4.5，氛围图用悠船 V8.1，草图用 5.0 Lite，修图用 Qwen Edit，组合使用效率最高。

4. **悠船系列提示词用英文**  
   中文容易丢细节，服装/姿态描述用英文或中英双语更稳。

5. **Lib Navo / Lib Image 暂时不适合高精度人物**  
   更适合概念图、场景图，人物面部和一致性不如 Seedream 和悠船。

---

## 六、一句话总结

> **电商人物/家居服主视觉：Seedream 4.5 或 4.6。**  
> **高质感氛围/品牌大片：悠船 V8.1。**  
> **快速草图/试稿：Seedream 5.0 Lite 或 Z-image Turbo。**  
> **修图改图：Qwen Edit。**  
> **动漫插画：悠船 Niji 7。**
