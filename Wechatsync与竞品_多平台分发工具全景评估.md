## 一、Wechatsync 是什么

**全称**：文章同步助手（Wechatsync）
**类型**：开源免费 Chrome 浏览器扩展
**GitHub**：5.7K Stars（截至2026年6月）
**官网**：wechatsync.com
**最新版本**：1.0.10（核心功能稳定，社区持续维护）

### 核心原理

依赖你浏览器里已登录的平台 Cookie/Token，直接从当前页提取正文 → 勾选目标平台 → 一键发布。所有数据存本地，不经过第三方服务器。

### 支持平台（29+）

| 类别 | 平台 |
|---|---|
| 主流自媒体 | 微信公众号、知乎、微博、头条号、百家号、搜狐号、大鱼号、一点资讯 |
| 技术社区 | CSDN、掘金、SegmentFault(思否)、博客园、51CTO、慕课网手记、开源中国 |
| 自建站 | WordPress、Typecho |
| 其他 | Bilibili、豆瓣、简书 |

### 三种使用场景

1. **公众号文章同步到其他平台** — 打开公众号文章 → 点扩展 → 勾选平台 → 同步
2. **Markdown 写作并同步** — 内置 Markdown 编辑器，写好一键分发，自动转各平台格式
3. **任意网页提取** — Safari 阅读模式算法提取正文，从任何网页同步到各平台

### 优点

- ✅ **开源免费** — 无任何付费墙
- ✅ **数据本地** — 不经过第三方服务器，账号信息安全
- ✅ **零部署** — 装个 Chrome 扩展即用
- ✅ **平台覆盖广** — 29+，涵盖技术社区和主流自媒体
- ✅ **格式自适应** — 自动适配各平台的兼容格式

### 缺点

- ❌ **Cookie 依赖** — 需定期重新登录（建议每周检查一次）
- ❌ **格式缺陷** — 简书/CSDN 上偶有格式错乱，复杂表格/代码高亮需手动调整
- ❌ **无多账号管理** — 不支持多账号矩阵操作
- ❌ **无数据监控** — 不提供发布统计、阅读分析
- ❌ **维护节奏** — 核心版本 1.0.10 较旧，部分平台改版后需社区适配
- ❌ **风控风险** — 批量同步可能触发平台反垃圾机制

## 五、对海风场景的建议

### 你的现状

- 核心阵地：公众号（已有自动管线）
- 自动同步：百家号（微信官方同步 ✅）、知乎（解禁后配置 ✅）
- 唯一未覆盖：今日头条（无官方自动同步）
- 未来可能扩展：搜狐号（GEO 引用率 #3）

### 推荐策略

| 需求 | 推荐工具 | 理由 |
|---|---|---|
| **当前：今日头条同步** | **Wechatsync** | 免费、零部署、只需要一个平台，装个扩展就搞定 |
| **未来：管理 8 个平台** | **蚁小二** | 50+ 平台覆盖、数据分析、多账号管理，成熟度最高 |
| **预算有限的新手方案** | **新榜小豆芽** | 免费版功能够用，上手最快 |
| **企业/多账号场景** | 融媒宝 / 聚媒通 | 当前不需要 |

### 最简方案

```
已有管线 → 公众号发文
    ↓ 自动同步（零操作）
    知乎（解禁后配） + 百家号（已配）
    ↓ Wechatsync（手动操作<1分钟）
    今日头条
    ↓ （暂不处理，观察期）
    其他平台（搜狐号等）
```

**一句话**：现在就装 Wechatsync，只用于今日头条这一个平台的同步即可。等平台扩张到 4 个以上再考虑蚁小二。
