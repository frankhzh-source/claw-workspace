链式推理 chain reasoning

因果链追踪：从架构设计一路传导到你的训练参数选择。

链 1：编码器链条（标注决定了 LoRA 的成败）

Mistral-3 (7B LLM) → 理解自然语言 > 标签 → 标注必须 40-100词句子 → 触发词出现 2-3次 → 触发词有效 + 泛化好 ✅
如果标注是标签列表（如"ohwx person, pink pajama, bedroom"），Mistral-3 无法建立触发词与视觉特征的语义关联 → 触发词完全无效。

链 2：架构链条（DiT 的梯度特性决定了 LR）

DiT (Transformer) 无下采样 → 梯度尺度小于 UNet → 需更高 LR 补偿 → LR 范围 1.2e-3~1.8e-3 → 与 SDXL 相差 10-15倍 🚨
用 SDXL 直觉（1e-4）调 FLUX.2 → 训练跑了 1200 步但权重几乎无变化 → 以为是步数不够 → 加到 2000 步还是无效 → 浪费数小时。

链 3：硬件的上限链条（24GB 决定了配置链）

5090D v2 24GB → 必须 quantize FP8 → low_vram offload CPU → gradient_checkpointing → batch_size=1 → VRAM 22-23.5GB 稳定 ✅
少开任何一个优化开关 → 训练到 300 步附近 OOM → 白跑数十分钟。

链 4：服装品类链条（数据多样性 → 泛化能力）

17个产品代码 → 每个品覆盖3种角度 → 摆拍/模特不同光源混合 → Mistral-3 学到泛化特征 → 多角度可生成 ✅
如果 17 个产品全是正面摆拍白底图 → LoRA 学到的不是"这个产品"而是"正面摆拍白底的画风"。

逆向反推 reverse: 从失败现象到根因

以你的具体配置（Klein 4B + 5090D v2 24GB + 服装LoRA）为上下文，从失败现象逆推根因。

#现象逆推路径根因定位修复
1 OOM，训练到 300 步崩溃 OOM → VRAM 超 24GB → quantize/low_vram 未同时开启 优化链不完整 quantize+low_vram+gradient_checkpointing+cache_latents 全开
2 1200 步跑完，触发词完全无效 触发词无效 → LoRA 权重近乎随机 → 梯度信号不足 → LR 太低（可能用了 < 5e-4） LR 低了 10 倍 LR 从 1.5e-3 起步，不要低于 1e-3
3 触发词有效，但生成图背景永远是白底 背景一致 → 训练图全是白底 → 标注未描述背景 → Mistral-3 把"白底"当成触发词特征 标注缺少背景描述 每条标注加场景/光线描述
4 不加触发词时，所有输出都带粉色 风格偏移 → 权重污染 → 标注中"pink"出现率 > 40% 高频词污染 训练前跑词频检查，平衡颜色描述分布
5 checkpoint-1000 不如 750 后面不如前面 → 过拟合 → 步数过多，LR 未衰减 checkpoint 倒挂 保留所有中间输出，选最佳而非最后；LR 衰减策略
6 Loss 不降反升（震荡） Loss 不降 → 梯度更新方向不稳定 → LR 过高或数据有噪声 LR 过高/标注错 LR 减半到 7.5e-4，检查标注是否矛盾
逆向反推核心原则
逆推时始终以你的配置为约束：看到"触发词无效"不要先怀疑模型，先查 LR 和 Rank（Klein 4B 的固有约束）。

演义推演核心发现
6 个最常见变更中，4 个会导致训练失败或质量下降。最危险的两个误操作：Rank 降到 16 和 关闭 quantize——都触发致命失败且无法在训练中恢复。

词源污染排除

污染类型表现对你的影响检测方法
触发词被稀释 "ohwx person" 被"Chinese woman" "pink pajama"等高频描述稀释注意力权重 触发词出现率不足，Mistral-3 无法建立稳定关联 grep 统计触发词出现率，确保每条 2-3 次

场景污染 所有标注都以"bedroom"开头，导致 LoRA 无法泛化到户外/棚拍 你的摆拍图是棚拍，模特图是卧室，必须混合描述 检查场景关键词分布，确保覆盖 bedroom/studio/indoor/outdoor
触发词拼写不一致 部分标注写"ohwx person"，部分写"ohwxPerson" Mistral-3 把它们当成两个不同概念，触发词效果减半 grep -r "ohwx" *.txt | sort | uniq -c，检查所有变体
排除结论
排除两种污染后，你的训练策略应聚焦在 3 个核心约束上：
① Klein 4B 的 LR 必须 ≥ 1e-3，② 24GB 要求 6 个优化开关全开，③ Mistral-3 要求标注必须是 40-100 词的自然语言句子且词频均衡。
