1 TCO七层成本结构 — 计算公式
总公式
Agent TCO = L1 + L2 + L3 + L4 + L5 + L6 + L7
每层：Lₙ = 单层月固定成本 + (单位消耗量 × 单价)
层级 成本项 计算方式 典型占比 标签
L1 模型调用费 月Token消耗 × 单价 ① 15-25% 唯一被算
L2 算力基础设施 GPU租赁/折旧 + 带宽 + 电费 10-15% 常被忽略
L3 数据准备与标注 标注人力 + 清洗工具 + 知识库搭建 15-25% 严重低估
L4 Agent开发与调优 Prompt工程 + 框架搭建 + 测试迭代 10-20% 部分被算
L5 系统集成 ERP/CRM对接 + API网关建设 10-15% 常被忽略
L6 运维监控 监控系统 + 告警 + 版本管理 5-10% 几乎未算
L7 安全合规 权限体系 + 审计 + 等保合规 5-10% 几乎未算
⚠ 只算Token单价来算ROI，误差可达4倍以上。L3-L7五层隐藏成本占总TCO的45-80%。
① Token单价 = 调用价 + 输出价。大模型厂商通常报输入价×3≈输出价。月均消耗需根据场景预估（对话类：每轮≈2000Token，代码类：每任务≈8000Token）
2 Agent ROI 核心公式
ROI公式
Agent ROI = (Agent产出价值 − Agent持有成本) / Agent持有成本 × 100%
展开式
Agent ROI = (∑ 工单数 × 单工单价值 − TCO月均) / TCO月均 × 100%
变量 定义 计算方式 参考值
产出价值 Agent完成的业务产出 完成工单数 × 单工单价值 ② 参照人类员工产值
持有成本 Agent月度TCO Σ(L1-L7)月均成本 2000-50000元/月
产能复用率 同时处理的任务线数 Agent任务线数 / 人类任务线数 ③ 100-1000x
产能利用率 实际运行时间占比 实际运行h / 总可用h 人类≈60% / Agent≈95%+
135x 杠杆推导（产能复用率示例）
假设：单Agent月持有成本=2000元，替代人类2天工作量（月薪1.5万×2/22≈1360元）
单条线ROI = (1360−2000)/2000 = −32%（负值！）
100条线ROI = (200×1360−2000)/2000 = 135x
→ 核心结论：Agent ROI的真正杠杆不在单任务提效，在规模化复用
② 单工单价值对标人类员工。例如：人工处理1单质检需30分钟×时薪50元=25元/单，则Agent单工单价值=25元。
③ 产能复用率是ROI的核⼼杠杆。人类1天只能处理1条任务线，Agent可以同时处理100-1000条。当复用率≥50条时，Agent ROI从负值翻转为正值。
