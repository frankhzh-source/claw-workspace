## L1 — 官方一手来源（最高权威）

### 1. LibTV CLI 官方文档
- **地址**：https://www.liblib.tv/zh/cli
- **权威等级**：⭐⭐⭐⭐⭐ 官方
- **覆盖内容**：
  - CLI 安装脚本（`libtv install`）
  - 全部子命令：login / workspace / project / node / group / upload / image / script / model
  - 画布节点类型完整 schema
  - 模型 schema 字段定义
  - 管道（NDJSON）输出
  - 主流 Agent 接入支持：Kimi Code/Claw、MiniMax、Trae、通义灵码、文心快码、腾讯云代码助手
- **学习建议**：安装后跑一遍 `libtv --help`，不需要逐条背命令，了解有哪些能力即可

### 2. LibTV 飞书官方文档库
- **地址**：https://liblibai.feishu.cn/wiki/Loxfw6XHziYRk0kKzdjcFfp9nhb
- **权威等级**：⭐⭐⭐⭐⭐ 官方

- **学习建议**：首次使用时通读全局，后续遇到问题先来这里搜

## L2 — 深度教程（高权威，可全文精读）

- **权威等级**：⭐⭐⭐⭐ 技术博客深度评测
- **篇幅**：长文（约5000字）
- **适合阶段**：初步上手后
- **核心里程碑**：
  - ✅ 无限画布基础操作（新建画布、节点操作、连接规则）
  - ✅ 5 大节点类型详解（文本/图片/视频/音频/脚本）

  - ✅ 视频增强：2/4/6 倍放大 + 帧率提升（30/60/90fps）
